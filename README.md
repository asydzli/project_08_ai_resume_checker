# Project 08 - AI Resume Checker

## Deskripsi

AI Resume Checker membantu menganalisis resume menggunakan Google Gemini AI.

---

## Cara Menjalankan

```bash
cd project_08_ai_resume_checker
```

```bash
streamlit run app.py
```

---

## Fitur

- Upload Resume PDF
- Upload Resume DOCX
- Analisis Resume
- ATS Friendly Checker
- Resume Score
- Saran Perbaikan

---

## Teknologi

- Python
- Streamlit
- Google Gemini AI
- python-dotenv
- pypdf
- python-docx
